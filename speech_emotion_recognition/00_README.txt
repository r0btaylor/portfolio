This is a machine learning project that attempts to regognise and classify emotion from recorded speech samples.
Current work is in progress.

Script 1 is an example of visualising spectral data in order to decided the most appropriate feature to employ in the project
Script 2 is an example of learning a machine learning algorithm for classification.

data folder contains required data for scripts to run:
		3 x example audio files for comparative visualisation
		2 x .csv containing extract MFCC raw and mean data
